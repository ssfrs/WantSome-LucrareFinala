1. Scopul proiectului

Scopul principal al proiectului este realizarea unui site/unei aplicații web de tip E-commerce ce se adresează în principal iubitorilor de artă și nu numai. Arta fiind de tip „Pictură”, „Artă Digitală”, „Sculptură” și orice obiect „Artizanat” cu o însemnătate deosebită din punct de vedere emoțional cât și al muncii din partea creatorului - Artistului.

Pentru o bună funționalitate, aplicția depinde atât de vânzători (artisti) cât și de cumpărători. Vânzătorii își pot „posta” produsele și în funcție de tipul „anunțului” cumpărat pentru un produs, acesta este afișat în diferite dimensiuni în pagina cu produse (Galerie). Totodată ei pot alege și modul în care să vândă opera: vânzare cu preț fix, cu preț negociabil, cu preț licitabil sau cu preț de tip ofertă.

Cumpărătorii pot salva produsele ca favorite și, bineînțeles, pot cumpăra produsul dorit sau produsele dorite. În cazul în care sunt mai multe produse se pot face achiziții DOAR DE LA ACELAȘI ARTIST.

Avatajele principale ale acestei aplicații, față de alte aplicații de pe piața actuală, sunt:
- posibilitatea de a vinde/cumpăra un produs în 4 tipuri de preț: Fix, Negociabil, Licitant, Tip Ofertă;
- ca vânzător, posibilitatea de a publica produsul în 4 dimensiuni, fiecare dimensiune având rol de scoatere în evidență a produsului respectiv.

Toate produsele adăugate cât și conturile create vor fi salvate într-o „bază de date”, în mometul de față, în localStore.

2. Tehnologii folosite

Pentru realizarea acestei aplicații am folosit următoarele:

- HTML;
- CSS;
- SCSS;
- JavaScript.

2.1. HTML

Hyper Text Markup Language (HTML)

HTML este limbajul standard de markup folosit pentru a defini structura de bază a unei pagini web. Nu este un limbaj de programare, ci un limbaj descriptiv cu care definim structura paginilor web.

2.2. CSS

Cascading Style Sheets (CSS)

CSS este un limbaj folosit pentru a crea foi de stiluri - stylesheets care descriu felul în care se prezintă vizual un document scris în HTML - de exemplu, pentru a schimba font-ul, culoarea sau mărimea textului, pentru a împărți conținutul în mai multe coloane sau pentru a adăuga animații ori alte particularități ce țin de cromatică -. Versiunea actuală a acestui limbaj este CSS3.

2.3. SASS

Syntactically Awesome Style Sheets (SASS)

CSS with superpowers

SASS este un preprocesor CSS, o unealtă care ne permite să generăm cod CSS pe baza unui cod scris într-o sintaxă specificată.

2.4. JavaScript

JavaScript (JS)

JavaScript este cel mai popular limbaj pentru scripturi şi funcţionează în toate browserele importante, cum ar fi Internet Explorer, Firefox, Chrome, Opera şi Safari.
- JavaScript a fost proiectat pentru a adăuga interactivitate paginilor HTML
- JavaScript este un limbaj pentru scripturi
- Un limbaj pentru scripturi este un limbaj de programare simplificat
- JavaScript este, în general, înglobat direct în paginile HTML
- JavaScript este un limbaj interpretat (adică scriptul este executat direct, fără compilare prealabilă).

2.5. GIT Hub

GIT Hub

GIT Hub este o platformă online ce se laudă cu:
- cel mai popular version control system;
- dezvoltat de catre Linus Torvalds, creatorul Linux
- este un sistem de versionare distribuit, ceea ce inseamna ca fiecare membru al unui repository care tine intreg codul corespunzator unei aplicatii, poate avea o copie locala a acestuia, complet functionala; acest lucru permite lucrul la distanta sau offline: schimbarile asupra codului se fac local pentru ca ulterior sa fie sincronizate cu cele din repository-ul principal, de pe server
- toate modificarile sunt stocate sub forma unui arbore, fiecare modificare este numita commit
